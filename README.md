# Goravel Document

This is vuepress skeleton for goravel document.

## Clone 

```
git clone --recursive git@github.com:goravel/docs-web.git
```

## Install

```
pnpm install
```

## Run dev

```
pnpm docs:dev
```

## Run Build

```
pnpm docs:build
```

## Run Deploy

```
pnpm docs:deploy
```
