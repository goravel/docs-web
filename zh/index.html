# 关于 Goravel

Goravel 是一个功能完备、具有良好扩展能力的 Web 应用程序框架。作为一个起始脚手架帮助 Golang 开发者快速构建自己的应用。

# 主要功能

  • 自定义配置
  • HTTP 服务
  • 用户认证
  • 数据库 ORM
  • 数据库迁移
  • 日志
  • 缓存
  • Grpc
  • Artisan 命令行
  • 任务调度
  • 队列
  • 事件系统
  • 邮件
  • Mock
